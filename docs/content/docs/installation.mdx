## Requirements

- **Python**: 3.10, 3.11, 3.12 or 3.13.
- **OS**: macOS, Linux, Windows.
- **One dependency**: [`httpx`](https://www.python-httpx.org/) `>=0.25`, installed automatically.

## Install from PyPI

<Tabs>
  <Tab title="pip">```bash pip install memsy ```</Tab>
  <Tab title="uv">```bash uv add memsy ```</Tab>
  <Tab title="poetry">```bash poetry add memsy ```</Tab>
  <Tab title="pipenv">```bash pipenv install memsy ```</Tab>
</Tabs>

## Verify the install

```python
import memsy
print(memsy.__name__)          # 'memsy'
from memsy import MemsyClient  # no ImportError
```

## Dev extras

If you're contributing to the SDK or want the test tooling locally:

```bash
pip install "memsy[dev]"
```

Adds `pytest`, `pytest-asyncio`, `anyio`, `black`, and `ruff`.
